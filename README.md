
# Computer Systems Lab – Hardware Investigation

## Overview
In this laboratory you will investigate computer hardware by:

- Disassembling a desktop computer
- Identifying internal components
- Documenting hardware with images
- Exploring Raspberry Pi architecture
- Reflecting on your learning

You will complete the activity using **GitHub Classroom**.

---

## Submission Steps

1. Accept the GitHub Classroom assignment.
2. Clone your repository:

```
git clone <your-repository-url>
cd computer-systems-lab
```

3. Complete all files in:

```
lab_journal/
```

4. Upload your images into:

```
images/
```

5. Commit and push your work:

```
git add .
git commit -m "Completed lab journal"
git push
```
